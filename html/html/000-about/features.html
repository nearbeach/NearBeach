NearBeach Features

Currently NearBeach has the ability to;

  • Setup Customer/Organisation information

  • Requirements/Requirement Items

  • Kanban Boards

  • Project/Tasks

  • Sprint Functionality

  • Request for Changes (RFC)

We recommend reading NearBeach Flow first to get an understanding of how each of the different modules will work with each other. Once you are ready to use NearBeach, please read the Basic Usage under this documentation.
