Welcome to NearBeachïƒ

What is NearBeach?ïƒ

NearBeach is an Open Source Project Management system built using VueJS and the Django framework. NearBeach.org. You can;

  • Record basic Organisation and customer data

  • Gather user requirements for tracking purposes

  • Build Kanban Boards

  • Create projects/tasks for deliverying

  • Plan out your development using Sprints

  • Setup Request for Changes (RFC) for deployments

Who are we?ïƒ

Welcome to NearBeach. We are a small community of developers trying to make tools to help you manage your projects. Why not join our community and we can build the tools that you require.
