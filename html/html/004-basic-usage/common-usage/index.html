Common Usage

This covers common usage within NearBeach. For example adding a group to an object. This is the same method no matter if you are on a project/task/requirement etc.

Some objects will not contain certain functionality, for example you can’t add/remove Users or groups from a requirement item.

Add Group To Object

  1. On the “Group and Users” tab, click on the “Add Group to {object}” button

  2. The “Add Group Wizard” will appear

  3. Select one or more groups from the drop down

  4. Click on the “Add Group(s)” button

Assign Customer To Object

  1. On the “Group and Users” tab, clic on the “Add User to {object}” button

  2. The “Add User Wizard” will appear

  3. Select one or more users from the drop down. Please note, a user must exist in the groups currently assigned to the object

  4. Click on the “Add User(s)” button

Linked Objects

Linked objects are other objects within NearBeach that currently;

  • Relates to the object

  • Is Blocked by the object

  • Is Currently Blocking the object

  • Is sub object of current object

  • Is parent object of the current object

  • Is the duplicate of the object

  • Is duplicate object of the current object

These are used to connect all the objects together in NearBeach.

Add Link to Current Object

  1. Click on the “Create new link” button, and the “Link Wizard” will appear

  2. Select the type of object you wish to link

    1. Project

    2. Requirement

    3. Requirement Item

    4. Task

  3. Select the type of link you would like

    1. Relates To

    2. Is Blocked By

    3. Is Blocking

    4. Sub Object

    5. Parent Object

    6. Duplicate of

    7. Has Duplicate

  4. System will search for all appropriate objects, and display them in the card format.

  5. Use the search or pages to filter for the data you require.

  6. Tick those appropriate. Changing pages, or applying a filter will NOT remove those ticked.

  7. Click on the “Save Changes” button at the bottom of the modal to add the links

Remove Link From Current Object

  1. Click on the trash icon on the link object

  2. The confirmation modal will appear

  3. If the connection is a “block” link, you may change this to a “related” link by clicking on the “Migrate Link” button

  4. Otherwise clicking on the “Delete” button will remove the link

Documentation

The documentation functionality gives you the ability to upload required information against a single object.

Uploading Documentation

Uploads the documentation to the blob storage configured in your account.

  1. Click on the “New Document/File” dropdown

  2. Click on the “Upload Document”

  3. The “Upload Document” wizard will appear.

  4. Click “Browser” to find a document on your local system

  5. Clicking next will bring you to the renaming screen. Please note, documents should have a unqiue name

  6. Click next will upload the document

Creating Links

Links are useful for attaching external documentation, or any useful information i.e sharepoint documentation

  1. Click on the “New Document/File” dropdown

  2. Click on the “Link” button

  3. The “Add Link Wizard” will appear - fill out appropriately and hit save

Creating Folders

Folders help you organise the documentation

  1. Click on the “new Document/File” dropdown

  2. Click on the “Folder” button

  3. The “Add Folder Wizard” will appear - fill out appropriate and hit save

Moving Documents/Links/Folders

Dragging and dropping the documents/links/folders will give you the ability to move objects around.

Customer Links

You will be able to see what current objects are connected to a customer by visiting the customer profile.

Adding Customers To Objects

  1. Click on the “Add Customer” button

  2. The “Add Customer Wizard” will appear

  3. Select the appropriate customer from the dropdown

  4. Click on “Save Changes” to add the current customer to this object

Remove Customers from Object

  1. Click on the trash logo next to the customer card

  2. A confirmation modal will appear

  3. Click “Yes” to remove the customer from this object

Misc

Assigning Tags To Objects

  1. Click on the “Add tag to {object}” button

  2. The “Add Tag Wizard” will appear

  3. Select the appropriate tags from the dropdown

  4. Click on “Add Tag” button to add them.

To remove any unwanted tags, click click on the trash icon

Creating Public Links for Objects

  1. Click on the “Add Public Link” button

  2. A new public link will be created right away

You have the ability to create as many public links as you need. You can deactivate them by unticking the “Is Active” tickbox

To delete any public links, just click on the trash icon and confirm the deletion.

Notes

Add Note to Object

  1. Click on the “Add Note” button

  2. The “Add Note Wizard” will appear

  3. Create an appropriate note for the object and click on “Save”

Edit Note

Users can only edit their own notes.

  1. Next to the note, click on the “Edit” button

  2. The “Edit Note Wizard” will appear

  3. Edit the note appropriately and click on the “Save” button

Delete Note

Users can only delete their own notes.

  1. Next to the note, click on the “Delete” button

  2. The confirmation wizard will appear, click on “Yes”
