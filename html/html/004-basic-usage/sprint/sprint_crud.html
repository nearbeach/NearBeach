Sprint

Creating A New Sprint

Sprints can only be created from the Project or Requirement objects. Other objects can be linked into sprints.

To navigate to a sprint, simply click on the sprint card.

Creating New Sprints

  1. Click on the “Create new sprint” button

  2. The “New Sprint Wizard” will appear

  3. Fill out the form appropriately and click on the “Save” button

  4. You will be redirected to the newly created sprint

Link Object to Sprint

If you require to add this object to an existing sprint

  1. Click on the “Add Object to Sprint” button

  2. The “Add Object to Sprint Wizard” will appear

  3. Select the sprint you want to add the current object too

  4. Click the save button
